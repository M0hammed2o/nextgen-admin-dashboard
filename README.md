# NextGen AI — Admin Dashboard

Super Admin control panel for the NextGen AI Platform.

## Features

- Business management (create, edit, suspend, unsuspend)
- Owner/manager account creation per business
- Usage & billing overview
- WhatsApp test message sending
- Daily limits configuration
- Audit log viewer

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

Runs on `http://localhost:8080` by default.

## API Connection

Connects to the Admin API at `http://localhost:8001`.

## Build

```bash
npm run build
```
